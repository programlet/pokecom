対応環境

実行方法
--

コードを転送後、つぎの手順に従って操作してください

1. `POKE &BFE03,&1A,&FD,&B,0,1,&7E:CALL &FFFD8`と操作して、マシン語エリアを確保します
2. `RUN`してください
